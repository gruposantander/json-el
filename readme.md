# JSON Expression Language

A very simple expression language.

Installation:

```shell
npm install @gruposantander/json-el
```

Basic usage example:

```js
const el = require('@gruposantander/json-el');

const compiled = el.compile({ eq: 'abc' });
const result1 = fn('abc'); // result1 is true
const result2 = fn('zyz'); // result2 is false
```
